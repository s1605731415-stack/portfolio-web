# Experimental Dark Portfolio Plan

1. Add a low-poly hero core component with GSAP pointer parallax and slow ambient motion.
2. Rewrite the homepage hero around huge editorial typography and the low-poly core.

5. Remove or reduce any floating UI that competes with the new first impression.
6. Verify with tests, lint, production build, browser screenshots, and production deployment smoke checks.
